Key Signatures

Key signatures are always added to the beginning of pointed measure.

To add a key signature

  • Select the data entry mouse mode in the toolbox (click on the button ). A key signature can not be selected if mouse is not in data entry mode!
  • In the toolbox select the clef/key/time tab
  • Then select the desired key signature.
  • Move the mouse cursor over the measure to which you want To add the key signature. Notice that the cursor changes into a pointing finger and that each measure highlights as you move the cursor over it.
  • Click the mouse To add the key signature at the beginning of the highlighted measure.

To remove a key signature

  • Select it with the mouse (click on it or drag around it).
  • Either, press delete key Del or click Cut either on contextual menu, on main menu, or in toolbar .
