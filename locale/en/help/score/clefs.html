Clefs

To add a clef

  • Select the data entry mouse mode in the toolbox (click on the button . A clef can not be selected if mouse is not in data entry mode!
  • In the toolbox select the clef/key/time tab
  • Then select the desired clef.
  • Move the mouse cursor over the point to which you want To add the clef. Notice that the cursor changes into a pointing finger and that each measure highlights as you move the cursor over it.
  • Click the mouse To add the cleft at pointed position in the highlighted measure.

To remove a clef

  • Select it with the mouse (click on it or drag around it).
  • Either, press delete key Del or click Cut either on contextual menu, on main menu, or in toolbar
