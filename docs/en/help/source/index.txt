LenMus Users' Guide
====================

**Target audience:**  People interested in learning how to use the Lenmus program. Although I've hardly tried to create an intuitive program, so that you will not have to read any documentation, there are certain areas, such as the score editor, that are very complex and documentation about how to do things is required.
